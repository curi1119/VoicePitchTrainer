# VoicePitchTrainer

発声の音程(ピッチ)を鍛えるトレーニングアプリ。アプリが再生した音と同じ高さの声をマイクに向かって出し、合っているかをリアルタイム判定します。PC・スマホのブラウザで動作します(マイクを使用)。

▶ **<https://curi1119.github.io/VoicePitchTrainer/>**

(開発初期に使った単一 HTML のプロトタイプも <https://curi1119.github.io/VoicePitchTrainer/prototype/> で動きます)

## 機能

- **チューナー(自由練習)**: 声の音程をアナログメーター・88鍵ピアノ・波形でリアルタイム表示
- **単音発声トレーニング**: 出題音を聞いて同じ高さの声を発声。±50セント以内を約1.5秒キープで正解。不正解後も正しい音が出せるまで練習を続けられる
- **音階練習**: ドレミファソ…のガイドに合わせて発声し、音ごとに○×判定。1周ごとに半音ずつ上がる
