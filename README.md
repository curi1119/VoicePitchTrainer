# ピッチラボ — Pitch Lab

発声の音程(ピッチ)を鍛えるトレーニングアプリ。アプリが再生した音と同じ高さの声をマイクに向かって出し、合っているかをリアルタイム判定します。PC・スマホのブラウザで動作します(マイクを使用)。

▶ **<https://curi1119.github.io/VoicePitchTrainer/>**

## 機能

- **チューナー(自由練習)**: 声の音程の軌跡をピッチグラフ(時間×音高)でリアルタイム表示。キーを設定するとスケール外の音を赤で表示
- **鍵盤**: 88鍵ピアノを画面いっぱいに表示して参照音を鳴らせる(スマホは縦型表示)。キー設定で基音・スケール外を色分け、度数表記(I〜VII)の表示も可能
- **単音発声トレーニング**: 出題音を聞いて同じ高さの声を発声。±50セント以内を約1.5秒キープで正解。不正解後も正しい音が出せるまで練習を続けられる
- **音階練習**: ガイドに合わせて発声し、音ごとに○×判定。1周ごとに半音ずつ上がる。7種のパターン(5音・1オクターブ・トライアド・アルペジオ・3度進行・半音階・3音)

音色はピアノ1(サンプル音源)・ピアノ2(ドライ)・ピアノ(合成)・ビープ音の4種から選択。ヘッダーの音量つまみで全体音量を調整できます。
